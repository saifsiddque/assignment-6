The difference between var let and const : Var, let, and const is the keyword to declare variables in javascript. Var variable is an old method to declare a variable in javascript. In modern javascript, we use the let and const variable, which is introduced in the ES2015(ES6) update; now, the let and const variable is used more frequently in modern javascript compared to the var variable. The scope of a var variable is functional scope. The scope of a let variable is block scope. The scope of a const variable is block scope. It can be updated and re-declared into the scope.
